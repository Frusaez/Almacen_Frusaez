📋 Guía: Subir la app a GitHub

Sigue estos pasos en orden. Al final tendrás una URL que puedes enviar por WhatsApp.
1Crear cuenta en GitHub (si no tienes)

Ve a github.com y pulsa Sign up.

Rellena: email, contraseña y nombre de usuario.
Elige un nombre sencillo, por ejemplo: frutosecossaez

Verifica tu email cuando llegue el correo de confirmación.

⚠️ Si ya tienes cuenta de GitHub, salta al Paso 2.
2Crear el repositorio

Una vez dentro de GitHub:

  • Pulsa el botón verde New (arriba a la izquierda)
  • En Repository name escribe: almacen
  • Selecciona Public
  • Deja todo lo demás sin tocar
  • Pulsa Create repository
3Subir los archivos

En la página del repositorio recién creado verás un enlace que dice "uploading an existing file". Púlsalo.

Se abre una pantalla donde puedes arrastrar archivos. Tienes que subir respetando las carpetas. Hazlo en este orden:

Primero — arrastra los archivos de la raíz:

index.html
manifest.json
sw.js

Pulsa Commit changes (botón verde abajo). Escribe como mensaje: archivos raíz.

Después — sube la carpeta css:
Pulsa Add file → Upload files → arrastra el archivo css/style.css.
GitHub detecta la barra y crea la carpeta automáticamente.
Commit: css.

Después — sube la carpeta js:
Arrastra js/app.js y js/config.js juntos.
Commit: js.

Después — sube la carpeta img:
Arrastra img/icon-192.png y img/icon-512.png.
Commit: img.

💡 Truco: puedes arrastrar carpetas enteras directamente en Chrome. Prueba arrastrando la carpeta css completa de una vez.
4Activar GitHub Pages (la URL pública)
  • En tu repositorio, pulsa Settings (pestaña arriba)
  • En el menú de la izquierda busca Pages
  • En Branch selecciona main
  • En la carpeta selecciona / (root)
  • Pulsa Save

Espera 1-2 minutos y recarga la página. Aparecerá un mensaje verde con tu URL:

https://frutosecossaez.github.io/almacen
✓ Esa es la URL que envías a los empleados. Ya está activa.
5Conectar con Google Apps Script

La URL de GitHub solo sirve para abrir la app. Los datos se guardan en Google.
Si aún no has conectado el Apps Script, abre js/config.js desde GitHub:

  • En el repositorio pulsa la carpeta js → config.js
  • Pulsa el icono del lápiz ✏️ (editar)
  • Cambia PEGAR_URL_APPS_SCRIPT_AQUI por tu URL de Apps Script
  • Pulsa Commit changes

La app se actualiza sola en 1 minuto. No hay que avisar a nadie ni reenviar nada.

✓ Desde ese momento todos los dispositivos guardan y leen datos en tiempo real.
6Enviar a los empleados

Copia tu URL y envíala por WhatsApp, email o como prefieras:

https://TU-USUARIO.github.io/almacen

El empleado abre el enlace en el móvil → Chrome pregunta si quiere instalarla → pulsa Instalar → queda como icono en la pantalla de inicio, igual que cualquier app.

📱 En iPhone: abrir en Safari → botón compartir ↑ → "Añadir a pantalla de inicio"
🔄 Cuando quieras cambiar algo en la app

No hay que avisar a nadie ni reenviar el enlace.

  • Ve a github.com → tu repositorio
  • Pulsa el archivo que quieres cambiar (ej: js/config.js)
  • Pulsa el lápiz ✏️ → edita → Commit changes
  • En 1-2 minutos todos los empleados ven la versión nueva al recargar
✓ Un solo enlace para siempre. Los datos nunca se pierden.
